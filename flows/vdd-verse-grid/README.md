# Verse Grid (VDD)

**What it is:** The red “chips” under an expanded chapter — verse numbers and ranges like `4-6` — on the phone Contents screen and on the tablet’s left pane next to sloka detail.

**Why a separate flow:** The full app layouts live in `vdd-bhagavadgita.book-layouts`. Verse grid has its own **Universal Verse Grid Format (UVGF)** so grouping, labels, selection, and taps stay consistent everywhere.

**Where to read:** `03-specifications.md` (format), `02-visual.md` (ASCII), `04-plan.md` (code tasks).
