A multi-platform mobile application for studying the Bhagavad Gita in various languages with deep Sanskrit analysis.

## About the Project

This project is a modern application for reading and studying the Bhagavad Gita. The main focus is on translation accuracy, interface usability, and offline access to the sacred text.

### Key Features
*   **Multilingual Support**: Supports multiple languages including English, Russian, German, Spanish, and Thai.
*   **Sanskrit**: Original Sanskrit verses with transliteration and word-by-word meaning.
*   **Offline Mode**: All data is stored locally using SQLite/Drift.
*   **Modern Stack**: Built with Flutter for Android, iOS, and other platforms.

## Technology Stack

*   **Framework**: [Flutter](https://flutter.dev) (Dart)
*   **Database**: [Drift](https://drift.simonbinder.eu/) (SQLite)
*   **Architecture**: Spec-Driven Development (SDD), Document-Driven Development (DDD).
*   **Data**: JSON structures for texts and vocabularies.

## Project Structure

*   `app/`: Flutter application source code.
*   `data/`: Bhagavad Gita texts, translations, schemas, and metadata.
*   `flows/`: Development workflow documentation and architectural decisions.
*   `scripts/`: Utility scripts for content auditing.
*   `legacy/`: Legacy versions of the project in Java and Swift.

## Supported Languages

The project supports a wide range of languages:
*   **Source**: Sanskrit (संस्कृतम्)
*   **Primary**: English, Russian, German, Spanish
*   **Translations**: Thai (ไทย), Korean (한국어), Chinese (简体/繁體), Japanese (日本語), and more.
