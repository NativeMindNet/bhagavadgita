# Bhagavad Gita Book (ภัควัทคีตา)

[![English](https://img.shields.io/badge/lang-en-blue.svg)](README_en.md)

แอปพลิเคชันมือถือแบบหลายแพลตฟอร์มสำหรับการศึกษาคัมภีร์ภัควัทคีตาในภาษาต่างๆ พร้อมการวิเคราะห์ภาษาสันสกฤตอย่างลึกซึ้ง

## เกี่ยวกับโปรเจกต์

โปรเจกต์นี้เป็นแอปพลิเคชันที่ทันสมัยสำหรับการอ่านและศึกษาคัมภีร์ภัควัทคีตา โดยเน้นที่ความถูกต้องของการแปล ความง่ายในการใช้งาน และการเข้าถึงข้อความศักดิ์สิทธิ์แบบออฟไลน์

### คุณสมบัติหลัก
*   **รองรับหลายภาษา**: รองรับหลายภาษารวมถึง ภาษาไทย, อังกฤษ, รัสเซีย, เยอรมัน และสเปน
*   **ภาษาสันสกฤต**: บทสวดภาษาสันสกฤตดั้งเดิมพร้อมการถอดเสียงและการแปลความหมายทีละคำ
*   **โหมดออฟไลน์**: ข้อมูลทั้งหมดถูกจัดเก็บไว้ในเครื่องโดยใช้ SQLite/Drift
*   **เทคโนโลยีที่ทันสมัย**: สร้างด้วย Flutter สำหรับ Android, iOS และแพลตฟอร์มอื่นๆ

## เทคโนโลยีที่ใช้

*   **Framework**: [Flutter](https://flutter.dev) (Dart)
*   **Database**: [Drift](https://drift.simonbinder.eu/) (SQLite)
*   **Architecture**: Spec-Driven Development (SDD), Document-Driven Development (DDD).
*   **Data**: โครงสร้าง JSON สำหรับเนื้อหาและพจนานุกรม

## โครงสร้างโปรเจกต์

*   `app/`: ซอร์สโค้ดของแอปพลิเคชัน Flutter
*   `data/`: เนื้อหาภัควัทคีตา, บทแปล, โครงสร้างข้อมูล (schema) และข้อมูลเมตา
*   `flows/`: เอกสารเวิร์กโฟลว์การพัฒนาและการตัดสินใจทางสถาปัตยกรรม
*   `scripts/`: สคริปต์ยูทิลิตี้สำหรับการตรวจสอบเนื้อหา
*   `legacy/`: เวอร์ชันเก่าของโปรเจกต์ในภาษา Java และ Swift

## ภาษาที่รองรับ

โปรเจกต์นี้รองรับภาษาต่างๆ มากมาย:
*   **ต้นฉบับ**: สันสกฤต (संस्कृतम्)
*   **ภาษาหลัก**: ไทย (ไทย), อังกฤษ, รัสเซีย, เยอรมัน, สเปน
*   **บทแปลอื่นๆ**: เกาหลี (한국어), จีน (简体/繁體), ญี่ปุ่น (日本語) และอื่นๆ
